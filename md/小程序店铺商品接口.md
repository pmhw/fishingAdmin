# 小程序 · 钓场店铺商品接口

基础路径：`/api/mini`，**无需登录**（与钓场列表/详情一致）。  
仅展示 **`fishing_venue.status = 1`** 的钓场，且本店商品 **`venue_product.status = 1`**、公共商品 **`product.status = 1`**。

---

## 1. 本店分类列表

`GET /api/mini/venues/{venue_id}/shop/categories`

**响应 `data`：**

```json
{
  "list": [
    { "id": 1, "name": "饵料", "sort_order": 0 }
  ]
}
```

仅返回 **`venue_shop_category.status = 1`** 的分类。

---

## 2. 本店商品列表

`GET /api/mini/venues/{venue_id}/shop/products`

| Query | 说明 |
|-------|------|
| `page` | 页码，默认 1 |
| `limit` | 每页条数，默认 10，最大 50 |
| `shop_category_id` | 可选。**不传**=全部分组；**`0`**=仅未分组（`shop_category_id` 为空或 0）；**正整数**=该分类下商品 |

**列表项字段：**

| 字段 | 说明 |
|------|------|
| `venue_product_id` | 本店商品主键，**详情接口路径用此 id** |
| `product_id` | 公共库 SPU id |
| `shop_category_id` | 本店分类 id，可 null |
| `shop_category` | `{ id, name }` 或 null |
| `name`, `intro`, `cover_image`, `unit` | 来自公共商品 |
| `min_price` | 当前可售规格中的最低售价（元），无规格时为 null |
| `sku_count` | 可售规格数量（本店 SKU 与公共 SKU 均启用） |

---

## 3. 商品详情（含规格）

`GET /api/mini/venues/{venue_id}/shop/products/{vp_id}`

- **`vp_id`** = 列表返回的 **`venue_product_id`**

**响应 `data`：**

```json
{
  "venue_product_id": 12,
  "shop_category": { "id": 1, "name": "饵料" },
  "product": {
    "id": 3,
    "name": "xxx",
    "intro": "",
    "cover_image": "/storage/...",
    "images": [],
    "detail": "",
    "unit": "件"
  },
  "skus": [
    {
      "id": 101,
      "product_sku_id": 55,
      "spec_label": "500ml×1袋",
      "price": 12.5,
      "stock": 30
    }
  ],
  "min_price": 12.5
}
```

- `skus[].id` 为 **`venue_product_sku` 主键**，下单时建议使用（若后续做订单接口）。
- 仅返回 **`venue_product_sku.status = 1`** 且 **`product_sku.status = 1`** 的规格；`stock` 为当前库存（整数）。

---

## 4. 小程序调用示例

```javascript
const base = 'https://你的域名/api/mini'
const venueId = 1

// 分类
wx.request({ url: `${base}/venues/${venueId}/shop/categories` })

// 商品列表（某分类）
wx.request({ url: `${base}/venues/${venueId}/shop/products`, data: { page: 1, limit: 10, shop_category_id: 2 } })

// 商品详情
const vpId = 12
wx.request({ url: `${base}/venues/${venueId}/shop/products/${vpId}` })
```

---

## 5. 数据库依赖

需已执行 `md/venue_shop_product.sql`（及如有旧库则 `md/venue_shop_category_migration.sql`），存在 `venue_shop_category`、`venue_product`、`venue_product_sku`、`product`、`product_sku` 等表。
